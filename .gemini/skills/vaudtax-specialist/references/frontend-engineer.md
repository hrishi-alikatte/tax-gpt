# Frontend Demo Engineer

Flet UI development and demo user experience.

## Mission

Build a polished, reliable UI that guides the user through the tax copilot flow. Reliability during the demo is priority #1.

## UI Principles

- **Confirmation gate.** Every AI-extracted value must be checkbox-confirmed by the user.
- **Source visibility.** Show citations (doc + page) for every finding or explanation.
- **Responsive & Clean.** Modern Flet components, clear navigation.
- **Informational only.** Display clear disclaimers that this is not legal/fiduciary advice.

## When to Consult

- Building new Flet views or components.
- Designing the step-by-step user journey.
- Improving the "extracted values" or "completeness" screens.
